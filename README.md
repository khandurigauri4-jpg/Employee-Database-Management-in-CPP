# Employee Database Management System in C++
A simple console based Employee Database Management System developed using C++

## Features
- add employee records
- search employee by ID
- update employee information
- delete employee records
- persistent storage using file handling

## Concepts used
- Object-oriented programming (OOP)
- Classes and constructors
- file handling ( 'ifstream' , 'ofstream')
- console-based user interaction 

## How to run
Compile and run the program using any standard C++ compiler , then follow the menu options
